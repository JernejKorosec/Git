# Javascript tutorial by the book


## Custom tools
[NodeMon](https://nodemon.io/)


## Installation

Go through tutorial on web page...

```bash
mkdir mysql-test
cd mysql-test
npm init -y
npm install mysql
npm install -g nodemon
npm install --save-dev nodemon
etc...

```
## Running application
node_modules\.bin\nodemon test.js





## How to run the app
On Linux
```bash
./node_modules/.bin/nodemon app.js
``` 
On Windows (cmd)
```cmd
\node_modules\.bin\nodemon app.js
```
